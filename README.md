# UPI Transaction Analysis Dashboard

## Project Overview

This project analyzes UPI transaction data using Microsoft Excel to understand the growth of digital payments in India. The dashboard provides insights into transaction volume, transaction value, and trends across different banks and time periods.

## Objectives

* Analyze the growth of UPI transactions
* Identify transaction trends over time
* Compare transaction values and volumes
* Provide visual insights into digital payment adoption

## Dashboard Features

* Monthly transaction trend analysis
* Total transaction value and volume
* Interactive filters and charts
* Key performance indicators (KPIs)

## Tools Used

* Microsoft Excel
* Excel / CSV dataset
* Data visualization techniques

## Key Insights

* Significant growth in UPI transactions over the years
* Increasing adoption of digital payments
* Seasonal patterns in transaction volume

## Files in This Repository

* `upi_transaction_analysis.xlsx` → Excel dashboard file
* `dataset.csv` → Raw dataset used for analysis
* `images/` → Dashboard preview screenshots
